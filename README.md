# Sindhudyog - Premium Konkan D2C E-commerce

A modern, high-end, production-ready D2C e-commerce platform for authentic Konkan food products.

## 🚀 Built With
- **Frontend**: Vite + React + TSX + Tailwind CSS + Framer Motion
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Payment**: Razorpay (Integration Placeholder)
- **Authenticity**: 100% Traditional Sindhudurg Flavors

## 📂 Project Structure
- `/frontend`: Modern React application with premium UI/UX.
- `/backend`: Scalable API with MongoDB models for Products and Orders.
- `/logo.png`: Original brand logo.

## 🛠️ Getting Started

### 1. Backend Setup
```bash
cd backend
npm install
# Update .env with your MongoDB URI
npm run seed # To populate initial product data
npm run dev
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## ✨ Key Features
- **Dynamic Marketplace**: Advanced filtering by categories and real-time search.
- **Premium Design**: High-fidelity UI using 'Playfair Display' serif and 'Outfit' sans-serif.
- **Cart & Checkout**: Global cart state with multi-step checkout (COD/UPI).
- **Admin Command Center**: Track revenue, manage orders, and update stock.
- **Brand Storytelling**: Immersive About Us section explaining the Konkan heritage.
